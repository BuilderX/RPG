# RPG

Created by http://qhduan.com

本项目是个人娱乐之作，希望制作一个自己觉得“顺心意”的游戏，以RPG为主要元素的2D游戏

### 开发环境

node.js 系统运行

create.js 系列：

easel.js 画图

tween.js 动画

sound.js 声音

preload.js 自动加载

地图使用tiled，网址在 http://www.mapeditor.org/
